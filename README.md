## Install

1. In the `root` directory:

```git
$ git clone https://github.com/danielef/hexo-theme-ochuunn.git themes/ochuunn
```

2. Change the `theme` property in the `config.yml` file.

```yml
# theme: landscape
theme: ochuunn
```
